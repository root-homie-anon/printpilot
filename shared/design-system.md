# PrintPilot Design System

## Typography

### Primary Fonts
- **Headings**: Inter (600/700 weight) — clean, modern, highly legible at all sizes
- **Body**: Nunito (400/600 weight) — friendly, rounded terminals, excellent for printable content
- **Accent/Decorative**: Playfair Display — for elegant niche products (wedding, luxury planners)

### Font Sizing Scale
- Page title: 28–32pt
- Section heading: 20–24pt
- Subsection heading: 16–18pt
- Body text: 11–13pt
- Caption/label: 9–10pt
- Fine print: 7–8pt

### Typography Rules
- Minimum body font size: 11pt for readability when printed
- Line height: 1.4–1.6x for body text, 1.2x for headings
- Letter spacing: 0 for body, +0.02em for uppercase labels
- Never use more than 3 font weights on a single page
- Headings should contrast body font (e.g., serif heading + sans-serif body, or bold vs regular weight)

## Spacing

### Page Margins (A4: 210mm x 297mm)
- Standard margins: 15mm all sides
- Generous margins (journals/planners): 20mm left/right, 18mm top/bottom
- Bleed-safe zone: keep critical content 5mm inside trim line

### Grid System
- Base unit: 4mm
- Column layouts: 1-col (full width), 2-col (equal or 60/40 split), 3-col (equal)
- Row spacing: 8mm between content blocks
- Section spacing: 16mm between major sections
- Element padding: 4mm internal padding for boxes, cells, and containers

### Spacing Rules
- Maintain consistent vertical rhythm — all spacing should be multiples of the base unit (4mm)
- Tables and trackers: minimum 8mm row height for comfortable handwriting space
- Checkbox/circle size: 5mm diameter, 4mm spacing between items
- Lined areas: 7–8mm line spacing for handwriting

## Color Palette

### Base Palette
- **Background**: #FFFFFF (pure white) or #FAFAFA (warm white)
- **Primary text**: #1A1A1A (near black — softer than pure black)
- **Secondary text**: #6B7280 (medium gray)
- **Light borders**: #E5E7EB (subtle gray)
- **Accent light**: #F3F4F6 (background tint for sections)

### Niche-Specific Palettes

#### Wellness / Self-Care
- Primary accent: #A7C7E7 (soft blue)
- Secondary accent: #C3B1E1 (lavender)
- Highlight: #FFDAB9 (peach)

#### Productivity / Business
- Primary accent: #2D3748 (dark slate)
- Secondary accent: #4A90D9 (professional blue)
- Highlight: #F6E05E (warm yellow)

#### Kids / Education
- Primary accent: #FC8181 (coral)
- Secondary accent: #68D391 (green)
- Highlight: #F6AD55 (orange)

#### Home / Organization
- Primary accent: #81E6D9 (teal)
- Secondary accent: #FBD38D (warm gold)
- Highlight: #FEB2B2 (soft pink)

#### Finance / Budget
- Primary accent: #48BB78 (money green)
- Secondary accent: #2D3748 (dark slate)
- Highlight: #E2E8F0 (light slate)

#### Wedding / Events
- Primary accent: #FED7E2 (blush pink)
- Secondary accent: #C6A97C (gold)
- Highlight: #E8D5B7 (champagne)

### Color Rules
- Maximum 3 accent colors per product, plus the base palette
- Text on colored backgrounds must meet WCAG AA contrast ratio (4.5:1 minimum)
- Use muted/pastel tones for large background areas — never saturated primaries
- Decorative elements: 20–40% opacity for subtle patterning
- Borders and dividers: always use light gray (#E5E7EB) unless niche palette overrides
- Never use pure black (#000000) for text — use #1A1A1A or darker niche color instead
